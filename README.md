This project is an Excel-based dashboard designed to help track, manage, and analyze consumer complaints. I built this tool to make it easier to monitor complaint status, identify delays, and understand overall performance in one place.


# Overview
The Consumer Complaint Tracker allows users to record complaint details, automatically highlight pending issues, and review insights through a clean dashboard. It is fully customizable and can be used by individuals or teams who handle customer feedback.

# Key Features
**Structured Complaint Log**
Tracks complaint date, category, status, and resolution time
Option to add assigned team or agent
Easy to filter and update

**Automatic Highlighting**
Overdue complaints are clearly marked
Pending or unresolved issues stand out
Makes follow-up easier

**Interactive Dashboard**
Shows total complaints
Status breakdown (open, closed, in progress)
Category-wise insights
Monthly trend
Average resolution time

**Visual Summary Charts**
Displays complaint trends
Shows performance patterns
Helps identify problem areas
